Создать приложение Vue состоящее из одного компонента,  формы регистрации нового пользователя.
Форма регистрации и карточка пользователя в макете PSD.
Обязательные условия:
1.	Только Vue, никаких сторонних библиотек по типу jquery.
2.	Верстка по BEM методологии.
3.	По мере ввода, данные из формы должны заноситься в хранилище данных (vuex), а не в локальное состояние компонента.
4.	Дата рождения -  не стандартный datepicker
5.	Страна, город –  не стандартный select
6.	По мере заполнения данных пользователем,
	необходимо отобразить введенные данные интерактивно в карточку пользователя над формой (Имя, фамилия, логин, почта)
7.	Если не выбрана страна, город выбрать нельзя. 
8.	При клике по кнопке регистрации, если какое-либо поле, отмеченное * не заполнено,
	отобразить сообщение о необходимости заполнить.
	Если все поля заполнены вывести сообщение о успешной регистрации.
	[сообщения в макете не нарисованы, можно вывести произвольно]
